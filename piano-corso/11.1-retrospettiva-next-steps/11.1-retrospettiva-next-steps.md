## SLIDE 9: Esercizio Consigliato

**Contenuto slide**:
```
ESERCIZIO CONSIGLIATO

Prendete un vostro progetto legacy e:

1. Create CLAUDE.md con le convenzioni
2. Scrivete test baseline sulle API
3. Provate a fare UN refactoring
   (es. estrarre un service)
4. Verificate con i test

Iniziate piccolo, un file alla volta.
I test vi proteggono dagli errori.
```

**Testo da leggere**:
"Per mettere in pratica: prendete un vostro progetto, create CLAUDE.md, scrivete qualche test, provate un piccolo refactoring. Non serve fare tutto insieme. Un file alla volta, test che vi proteggono."
